# Tutor Website

This project is a practice website called **Tutor Website**, created to improve my front-end web development skills using **HTML**, **CSS**, and **JavaScript**.

## Learning Objectives

- Practice building a simple website layout using **HTML** and **CSS**  
- Strengthen understanding of **CSS Flexbox** and **CSS Grid** for responsive design  
- Practice **JavaScript DOM manipulation (DOM.js)** to make the page interactive  
- Improve coding workflow with **Git** and **GitHub**

## Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid)  
- JavaScript (DOM manipulation)

## Project Goals

- Understand how to structure a web project  
- Learn to design responsive layouts with CSS  
- Practice connecting HTML, CSS, and JS together  
- Develop problem-solving skills through front-end coding
